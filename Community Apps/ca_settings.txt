Hide Deprecated Applications=Скрыть устаревшие приложения
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=Этот параметр скрывает устаревшие приложения из списка доступных приложений. Устаревшие приложения по-прежнему будут отображаться в разделах установленных и предыдущих приложений.
Hide Incompatible Applications=Скрыть несовместимые приложения
This setting will hide applications that are listed as being incompatible with your version of Unraid=Этот параметр скроет приложения, которые указаны как несовместимые с вашей версией Unraid.
Allow install of second instance=Разрешить установку второго экземпляра
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=Этот параметр включает еще одну кнопку на карточках и всплывающих окнах, которые позволят вам переустановить установленное приложение, используя его значения шаблона по умолчанию. Этот параметр может время от времени сбивать с толку и обычно используется только в том случае, если вы хотите установить и запустить приложение дважды одновременно.
Show descriptions on application cards=Показывать описания на карточках приложений
This option dictates whether the descriptions appear on the cards or not=Этот параметр определяет, будут ли описания отображаться на карточках или нет.
Allow CA to check for updates to applications=Разрешить ПС проверять наличие обновлений для приложений
This setting enables CA to check for updates to applications Note that Community Applications will always check for updates to itself=Этот параметр позволяет ПС проверять наличие обновлений для приложений. Обратите внимание, что приложения сообщества всегда будут проверять наличие обновлений для себя.
Allow CA to send any emergency notifications=Разрешить ПС отправлять любые экстренные уведомления
This setting allows CA to periodically scan your system and alert you to the presence of=Этот параметр позволяет ПС периодически сканировать вашу систему и предупреждать вас о наличии
1 Known malware and security vulnerabilities within applications=1 Известных вредоносных программ и уязвимостей безопасности в приложениях
2 Critical security vulnerabilities within Unraid OS=2 Критических уязвимостях в системе безопасности Unraid OS

Save CA debugging information=Сохранить отладочную информацию ПС
Only enable this when advised to from the author of Community Applications=Включайте это только по рекомендации автора приложений сообщества.
