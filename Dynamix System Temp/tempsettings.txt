:temp_drivers_plug:
> Используйте **обнаружить** для поиска доступных драйверов с помощью сценария *sensors-detect*. После обнаружения приступайте к сохранению списка.<br>
> Используйте **сохранить**, чтобы сохранить список драйверов на диск, поле драйверов можно отредактировать перед сохранением (например, добавить неуказанный драйвер). Сохранение обязательно.<br>
> Используйте **загрузить драйверы** / **выгрузить драйверы»**, чтобы вручную загрузить или выгрузить доступные модули драйверов.
:end

:temp_processor_plug:
> Выберите один из доступных датчиков в качестве индикатора температуры процессора.
> Выходные данные этого датчика показаны внизу страницы.
> Выберите **Не используется**, чтобы скрыть отображение этого датчика.
:end

:temp_mainboard_plug:
> Выберите один из доступных датчиков в качестве индикатора температуры материнской платы.
> Выходные данные этого датчика показаны внизу страницы.
> Выберите **Не используется**, чтобы скрыть отображение этого датчика.
:end

:temp_fan_plug:
> Select one of the available probes as the array fan speed indicator.
> The output of this probe is shown at the bottom of the page.
> Select 'Not used' to suppress display of this probe.
:end

:temp_rescan_plug:
> **Повторное сканирование** обновляет список выбора и может использоваться, когда доступны новые датчики, например: установлен новый датчик кулера.
:end
